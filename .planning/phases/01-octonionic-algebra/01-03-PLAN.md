2. Implement `src/octonion/_operations.py`:
   - `octonion_exp(o: Octonion) -> Octonion`:
     - Split into real part `a` and imaginary vector `v`
     - `||v||` = norm of imaginary part
     - If `||v|| < epsilon`: return `exp(a) * identity` (pure real exponential)
     - Otherwise: `exp(a) * (cos(||v||) * e_0 + sin(||v||) * v/||v||)`
     - Works on [..., 8] batched tensors
   - `octonion_log(o: Octonion) -> Octonion`:
     - `||o||` = norm of full octonion
     - `a` = real part, `v` = imaginary vector, `||v||` = norm of imaginary part
     - If `||v|| < epsilon`: return `log(||o||) * e_0` (pure real log)
     - Otherwise: `log(||o||) * e_0 + (arccos(a/||o||) / ||v||) * v`
     - Returns pure or general octonion depending on input
   - `commutator(a: Octonion, b: Octonion) -> Octonion`:
     - Returns `a*b - b*a` (simple definition)
   - `inner_product(a: Octonion, b: Octonion) -> torch.Tensor`:
     - Returns `Re(a.conjugate() * b)` which equals `torch.sum(a.components * b.components, dim=-1)`
     - Actually, Re(a* * b) = sum of component-wise products (for real inner product on R^8)
   - `cross_product(a: Octonion, b: Octonion) -> Octonion`:
     - 7D cross product on the imaginary parts
     - `cross(a,b) = (a*b - b*a) / 2` restricted to imaginary part? No -- the 7D cross product for pure octonions is: `a x b = Im(a* * b)` for pure imaginary octonions, or equivalently `(a*b - b*a)/2` for pure octonions
     - More precisely: for pure imaginary octonions a, b: `a x b = Im(a*b)` = imaginary part of product
     - Return a PureOctonion (real part zero)
   - All functions also accept raw tensors [..., 8] for flexibility
