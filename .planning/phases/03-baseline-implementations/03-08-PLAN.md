**Extend find_matched_width to handle conv2d topology.**

Currently `find_matched_width` raises `NotImplementedError` for any topology other than "mlp". For conv2d, it needs to build `AlgebraNetwork` instances instead of `_SimpleAlgebraMLP` for param counting.

Key difference from MLP matching: For MLP, `find_matched_width` returns the algebra-unit hidden width (passed directly to `_SimpleAlgebraMLP`). For conv2d, it should return the `base_hidden` value (base filter count) for `NetworkConfig`. The binary search builds `AlgebraNetwork(NetworkConfig(algebra=algebra, topology="conv2d", depth=depth, base_hidden=candidate, input_dim=input_dim, output_dim=output_dim))` and counts params.

Implementation:

```python
def find_matched_width(
    target_params, algebra, topology, depth, tolerance=0.01,
    input_dim=784, output_dim=10, **kwargs
) -> int:
    if topology == "mlp":
        # existing MLP logic unchanged
        ...
    elif topology == "conv2d":
        from octonion.baselines._network import AlgebraNetwork
        from octonion.baselines._config import NetworkConfig

        lo, hi = 1, 512  # base_hidden for conv is typically smaller
        best_width, best_diff = lo, float("inf")
        while lo <= hi:
            mid = (lo + hi) // 2
            config = NetworkConfig(
                algebra=algebra, topology="conv2d", depth=depth,
                base_hidden=mid, input_dim=input_dim, output_dim=output_dim,
                activation=kwargs.get("activation", "split_relu"),
                output_projection=kwargs.get("output_projection", "flatten"),
                use_batchnorm=kwargs.get("use_batchnorm", True),
            )
            model = AlgebraNetwork(config)
            count = sum(p.numel() for p in model.parameters() if p.requires_grad)
            diff = abs(count - target_params) / target_params
            if diff < best_diff:
                best_diff = diff
                best_width = mid
            if diff <= tolerance:
                return mid
            elif count < target_params:
                lo = mid + 1
            else:
                hi = mid - 1
        if best_diff > tolerance:
            raise ValueError(...)
        return best_width
    else:
        raise NotImplementedError(f"Topology {topology!r} not supported")
```
