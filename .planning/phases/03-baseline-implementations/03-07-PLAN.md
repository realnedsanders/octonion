**Step 2: Implement ResidualBlock and replace `_build_conv`.**

Replace the current `_build_conv` method in `AlgebraNetwork` with a ResNet-style architecture. Do NOT remove or rename the existing conv support -- replace the internals.

Architecture design (following Gaudet/Maida 2018 and standard CIFAR ResNets):

```
3 stages with increasing filter counts:
  Stage 1: base_filters channels (e.g., 16 for base_hidden=16)
  Stage 2: base_filters * 2 channels
  Stage 3: base_filters * 4 channels

Each stage contains N residual blocks where N = depth // 3 (distribute evenly).
Each residual block: conv3x3 -> BN -> activation -> conv3x3 -> BN -> (+skip) -> activation

Downsampling: stride=2 on first conv of first block in stage 2 and stage 3 only.
Skip connection: identity if dimensions match, otherwise 1x1 conv + BN for channel/spatial mismatch.

After all stages: global average pooling -> fc_hidden -> activation -> output projection.
```

Key implementation details:
- `_ResidualBlock` must use `network._get_conv2d()`, `network._get_bn()`, `network._get_activation()` factory methods so it inherits the correct algebra-specific layers
- For the 1x1 shortcut conv, call `network._get_conv2d(in_ch, out_ch, kernel=1, padding=0)` -- this requires the conv layers to accept `stride` kwarg
- The block must handle both real tensors `[B, ch, H, W]` and hypercomplex tensors `[B, ch, dim, H, W]` for stride and skip connections
- BN reshape logic (real vs hypercomplex permute/reshape) from the current `_forward_conv` should be factored into a helper method so ResidualBlock can reuse it

**Step 3: Replace `_forward_conv`.**

The new forward should be simpler:
```
input_conv -> input_bn -> input_act
-> stage1 blocks -> stage2 blocks (first block stride=2) -> stage3 blocks (first block stride=2)
-> global average pooling
-> fc_hidden -> fc_act -> output projection
```

**Step 4: Distribute depth across stages.**

Use even distribution: `blocks_per_stage = [depth // 3, depth // 3, depth - 2*(depth//3)]`. For depth=28: [9, 9, 10]. For depth=3: [1, 1, 1].
