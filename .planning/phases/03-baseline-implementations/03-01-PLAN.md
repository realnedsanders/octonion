**_initialization.py**: Per-algebra weight initialization:
- `real_init(weight: Tensor, criterion: str = "he")`: Kaiming/He init via `nn.init.kaiming_normal_`.
- `complex_init(W_r: Tensor, W_i: Tensor, criterion: str = "glorot")`: Trabelsi et al. Rayleigh magnitude + uniform phase. sigma = 1/sqrt(2*(fan_in+fan_out)) for glorot, 1/sqrt(2*fan_in) for he. Sample magnitude from Chi2(df=2).sqrt() * sigma, phase from Uniform(-pi, pi). W_r = magnitude * cos(phase), W_i = magnitude * sin(phase).
- `quaternion_init(W_r, W_i, W_j, W_k: Tensor, criterion: str = "glorot")`: Parcollet/Gaudet polar form. sigma computed same way. Magnitude from Chi2(df=4).sqrt() * sigma. Three uniform phase angles. Decompose via: W_r = mag*cos(phi1), imag_mag = mag*sin(phi1), W_i = imag_mag*cos(phi2), remaining = imag_mag*sin(phi2), W_j = remaining*cos(phi3), W_k = remaining*sin(phi3).
- `octonion_init(weights: list[Tensor], criterion: str = "glorot")`: Extension of quaternion pattern to 8-DOF. Chi2(df=8).sqrt() for magnitude, 7 phase angles for polar decomposition. sigma = 1/sqrt(2*(fan_in+fan_out)) for glorot.

All init functions use `torch.no_grad()` context and modify tensors in-place via `.copy_()`.

**__init__.py**: Export all public names. Initially just config and init. Will grow as modules are added.

**_algebra_linear.py**: Four nn.Module linear layers following the existing OctonionLinear convention of [..., dim] tensor shapes:

`RealLinear(in_features, out_features, bias=True, dtype=float32)`:
- Wraps `nn.Linear(in_features, out_features, bias, dtype)`.
- Init: Kaiming/He via `real_init`.
- Forward: `x` shape [..., in_features] -> [..., out_features].

`ComplexLinear(in_features, out_features, bias=True, dtype=float32)`:
- Parameters: W_r, W_i as [out_features, in_features], bias_r, bias_i as [out_features] (if bias=True).
- Init: `complex_init(W_r, W_i, criterion='glorot')`.
- Forward: x shape [..., in_features, 2]. Extract x_r = x[..., 0], x_i = x[..., 1]. Compute: out_r = F.linear(x_r, W_r) - F.linear(x_i, W_i), out_i = F.linear(x_r, W_i) + F.linear(x_i, W_r). Stack + add bias.

`QuaternionLinear(in_features, out_features, bias=True, dtype=float32)`:
- Parameters: W_r, W_i, W_j, W_k as [out_features, in_features], bias 4-component [out_features, 4] (if bias=True).
- Init: `quaternion_init(W_r, W_i, W_j, W_k, criterion='glorot')`.
- Forward: x shape [..., in_features, 4]. Hamilton product expanded into 16 terms. Use F.linear for each component interaction. Cross-reference sign pattern with verified `Quaternion.__mul__` in `_tower.py`.

`OctonionDenseLinear(in_features, out_features, bias=True, dtype=float32)`:
- Parameters: 8 weight matrices W_0..W_7 each [out_features, in_features], bias [out_features, 8] (if bias=True).
- Init: `octonion_init([W_0, ..., W_7], criterion='glorot')`.
- Forward: x shape [..., in_features, 8]. Full octonionic matrix-vector product using STRUCTURE_CONSTANTS: for output component k, compute sum over i,j of C[i,j,k] * F.linear(x[..., i], W_j). This is equivalent to constructing an 8-component weight octonion for each (in, out) pair and multiplying.
- NOTE: This is NOT the rank-1 OctonionLinear (a*x)*b. This is a full linear map with 8*in*out real parameters.

**_param_matching.py**:
- `find_matched_width(target_params: int, algebra: AlgebraType, topology: str, depth: int, tolerance: float = 0.01, **kwargs) -> int`: Binary search for hidden width. Builds a temporary model at each candidate width, counts params, converges within tolerance. Raise ValueError if impossible.
- `param_report(model: nn.Module) -> list[dict]`: Iterate model.named_parameters(), return per-layer breakdown: [{"name": ..., "shape": ..., "real_params": ..., "pct": ...}].
