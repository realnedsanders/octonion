**_trainer.py**: Complete training utility.

`seed_everything(seed: int)`:
- Set `torch.manual_seed(seed)`, `torch.cuda.manual_seed_all(seed)`, `np.random.seed(seed)`, `random.seed(seed)`.
- Do NOT set `torch.backends.cudnn.deterministic = True` (performance over bit-exactness per CONTEXT.md).

`train_model(model, train_loader, val_loader, config: TrainConfig, output_dir: str, device: str = "cuda") -> dict`:
- **Setup**: Build optimizer via `_build_optimizer(model, config)` dispatching on config.optimizer ("adam" -> Adam, "sgd" -> SGD, "adamw" -> AdamW). Build scheduler via `_build_scheduler(optimizer, config)` dispatching on config.scheduler ("cosine" -> CosineAnnealingLR, "step" -> StepLR, "plateau" -> ReduceLROnPlateau). Create AMP GradScaler if config.use_amp. Create TensorBoard SummaryWriter in output_dir.
- **LR warmup**: For the first warmup_epochs, linearly scale LR from 0 to config.lr. Implement as a LambdaLR that returns `min(1.0, epoch / max(1, warmup_epochs))` composed with the main scheduler.
- **Training loop** (per epoch):
  1. `model.train()`, iterate train_loader batches.
  2. Forward pass wrapped in `torch.amp.autocast('cuda', enabled=config.use_amp)`.
  3. Loss computation (cross-entropy for classification -- accept loss_fn as parameter with default `nn.CrossEntropyLoss`).
  4. `scaler.scale(loss).backward()`, `scaler.step(optimizer)`, `scaler.update()`, `optimizer.zero_grad()`.
  5. Accumulate train loss.
  6. After epoch: compute gradient statistics. Iterate `model.named_parameters()`, collect `p.grad.norm()` for params with gradients. Log to TensorBoard: Grad/norm_mean, Grad/norm_max, Grad/norm_var.
  7. Log VRAM: `torch.cuda.max_memory_allocated() / 1e6` if CUDA available.
  8. Validation: call `evaluate(model, val_loader, device)`.
  9. Log to TensorBoard: Loss/train, Loss/val, Accuracy/val, LR.
  10. Scheduler step.
  11. Early stopping check: track best_val_loss, patience counter.
  12. Checkpoint if epoch % checkpoint_every == 0.
  13. Wall-clock timing per epoch.
- **Graceful shutdown**: Register SIGINT handler at start of training. On SIGINT, save checkpoint with "_interrupted" suffix, then raise KeyboardInterrupt.
- **Return**: dict with keys: train_losses (list), val_losses (list), val_accuracies (list), best_val_acc (float), best_val_loss (float), total_time_seconds (float), epochs_trained (int), early_stopped (bool).

`evaluate(model, loader, device) -> tuple[float, float]`:
- Return (loss, accuracy) for classification tasks.
- model.eval(), torch.no_grad() context.

`save_checkpoint(path, model, optimizer, scheduler, epoch, best_val_loss, metrics)`:
- Save full state dict to path.

`load_checkpoint(path, model, optimizer, scheduler) -> dict`:
- Load and restore all state. Return metadata dict with epoch, best_val_loss, metrics.

`run_optuna_study(model_builder_fn, train_loader, val_loader, algebra: AlgebraType, n_trials: int = 50, study_name: str = "hp_search", output_dir: str = "experiments/optuna", device: str = "cuda") -> dict`:
- Create an Optuna study with `optuna.create_study(direction="minimize", study_name=study_name)`.
- Define objective function that:
  1. Suggests hyperparameters via `trial.suggest_*`:
     - lr: `trial.suggest_float("lr", 1e-4, 1e-1, log=True)`
     - weight_decay: `trial.suggest_float("weight_decay", 1e-6, 1e-2, log=True)`
     - optimizer: `trial.suggest_categorical("optimizer", ["adam", "adamw", "sgd"])`
     - scheduler: `trial.suggest_categorical("scheduler", ["cosine", "step", "plateau"])`
     - batch_size: `trial.suggest_categorical("batch_size", [64, 128, 256])`
     - (Optionally) gradient_clip: `trial.suggest_float("gradient_clip", 0.0, 5.0)` -- per CONTEXT.md, clipping is an Optuna search parameter
  2. Builds a TrainConfig from suggested params (with reduced epochs for search, e.g., epochs=20)
  3. Builds model via model_builder_fn(algebra)
  4. Calls train_model and returns best_val_loss as the objective value
  5. Reports intermediate values via `trial.report()` for pruning
  6. Uses `optuna.pruners.MedianPruner` for early stopping of bad trials
- After study completes, return dict with: best_params (dict), best_value (float), n_trials (int), study_name (str).
- Save study results to `{output_dir}/{study_name}_results.json`.
- This implements the CONTEXT.md decision: "Hyperparameter search: Optuna integration for Bayesian optimization" and "LR scaling heuristic from Phase 2 NOT auto-applied -- let Optuna search find optimal LR for each algebra".

**DDP-ready**: Use `model.module` if wrapped in DDP. Do not add DDP wrapper here -- let caller handle that. But ensure all save/load code handles both DDP-wrapped and unwrapped models.

**_stats.py**: Statistical testing module.

`paired_comparison(results_a: list[float], results_b: list[float]) -> dict`:
- Compute paired t-test via `scipy.stats.ttest_rel`.
- Compute Wilcoxon signed-rank via `scipy.stats.wilcoxon`.
- Compute Cohen's d effect size.
- Compute 95% confidence interval for mean difference.
- Return dict: {"t_stat", "t_p_value", "w_stat", "w_p_value", "effect_size", "mean_diff", "ci_lower", "ci_upper"}.

`cohen_d(a: list[float], b: list[float]) -> float`:
- Standard Cohen's d: (mean(a) - mean(b)) / pooled_std.

`holm_bonferroni(p_values: list[float], alpha: float = 0.05) -> list[dict]`:
- Sort p-values, apply Holm-Bonferroni step-down correction.
- Return list of dicts with original_p, adjusted_p, rejected (bool).

`confidence_interval(data: list[float], confidence: float = 0.95) -> tuple[float, float]`:
- Return (lower, upper) bounds using t-distribution (handles small N).

**_plotting.py**: Plotting utilities using matplotlib/seaborn.

`plot_convergence(metrics: dict, output_path: str)`:
- Plot train_losses and val_losses vs epoch on same axes.
- Optional val_accuracies on secondary y-axis.
- Save to output_path as PNG.

`plot_comparison_bars(results: dict[str, list[float]], metric_name: str, output_path: str)`:
- Bar chart with one bar per algebra (R, C, H, O).
- Error bars from standard deviation across seeds.
- Seaborn styling.

`plot_param_table(param_reports: dict[str, list[dict]], output_path: str)`:
- Formatted table showing per-algebra parameter counts.
- Columns: Layer, R params, C params, H params, O params.
- Save as PNG using matplotlib table.

All plot functions use `plt.savefig(path, dpi=150, bbox_inches='tight')` and `plt.close()` (do not leave figures open).
