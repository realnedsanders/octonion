   - All inputs detached from autograd graph (operate on pure values, not graph nodes).

**Multiplication** (`jacobian_mul`):
```python
def jacobian_mul(a, b) -> tuple[torch.Tensor, torch.Tensor]:
    # J_a[k, i] = sum_j C[i,j,k] * b[j] -- this is R_b transposed
    J_a = torch.einsum("ijk, ...j -> ...ki", C, b)
    # J_b[k, j] = sum_i C[i,j,k] * a[i] -- this is L_a transposed
    J_b = torch.einsum("ijk, ...i -> ...kj", C, a)
    return J_a, J_b
```

**Exponential** (`jacobian_exp`): Derive from exp(a + v) = exp(a)(cos||v|| + sin||v||/||v|| * v_hat). The 8x8 Jacobian has block structure:
- Row 0 (real output): d(exp(a)cos||v||)/da = exp(a)cos||v||; d(exp(a)cos||v||)/dv_i = -exp(a)sin(||v||)*v_i/||v||
- Rows 1-7 (imag output k): d(exp(a)sin(||v||)*v_hat_k)/da = exp(a)sin(||v||)*v_hat_k; d/dv_i involves chain rule through ||v|| and v_hat
Handle near-zero ||v|| with L'Hopital: sin(||v||)/||v|| -> 1, derivative of sin(||v||)/||v|| -> 0 as ||v|| -> 0.

**Logarithm** (`jacobian_log`): Derive from log(q) = log(||q||) + arccos(a/||q||)/||v|| * v. Block structure analogous to exp.

**Conjugate** (`jacobian_conjugate`): Trivial diagonal matrix diag([1, -1, -1, -1, -1, -1, -1, -1]).

**Inverse** (`jacobian_inverse`): For f(o) = conj(o)/||o||^2. Use quotient rule:
J[k, i] = (diag_conj[k,i] * ||o||^2 - conj(o)_k * 2*o_i) / ||o||^4
where diag_conj[k,i] = +1 if k=i=0, -1 if k=i>0, 0 otherwise.

**Inner product** (`jacobian_inner_product`): For f(a, b) = sum_i a_i * b_i (scalar output), J_a = b^T (shape [..., 1, 8]), J_b = a^T.

**Cross product** (`jacobian_cross_product`): Uses octonion_mul on pure imaginary parts. J_a and J_b derivable from mul Jacobian applied to the pure-imaginary projections. Note: only rows 1-7 of output are non-zero (pure imaginary result).

All Jacobians operate on raw [..., 8] tensors, not Octonion instances. Use `STRUCTURE_CONSTANTS.to(device=..., dtype=...)` consistently.
