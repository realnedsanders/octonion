```python
_SHORT_TO_ALGEBRA: dict[str, AlgebraType] = {a.short_name: a for a in AlgebraType}

def _algebra_from_shortname(name: str) -> AlgebraType:
    if name not in _SHORT_TO_ALGEBRA:
        raise ValueError(f"Unknown algebra short name: {name!r}. Valid: {list(_SHORT_TO_ALGEBRA.keys())}")
    return _SHORT_TO_ALGEBRA[name]
```

2. **Directory discovery**: Walk `results_dir/{task}/{optimizer}/{algebra}/seed_{N}/` to discover completed runs. Do NOT require all tasks/algebras -- work with whatever exists.
```python
def _discover_runs(results_dir: str) -> list[dict]:
    """Discover all completed experiment runs from directory structure."""
    runs = []
    results_path = Path(results_dir)
    for task_dir in sorted(results_path.iterdir()):
        if not task_dir.is_dir(): continue
        for opt_dir in sorted(task_dir.iterdir()):
            if not opt_dir.is_dir(): continue
            for alg_dir in sorted(opt_dir.iterdir()):
                if not alg_dir.is_dir(): continue
                for seed_dir in sorted(alg_dir.iterdir()):
                    if not seed_dir.is_dir() or not seed_dir.name.startswith("seed_"): continue
                    result_file = seed_dir / "result.json"
                    if result_file.exists():
                        runs.append({
                            "task": task_dir.name,
                            "optimizer": opt_dir.name,
                            "algebra": alg_dir.name,
                            "seed": int(seed_dir.name.split("_")[1]),
                            "seed_dir": seed_dir,
                            "result_file": result_file,
                        })
    return runs
```

3. **Hessian analysis**: For each seed in `hessian_seeds` that has checkpoint files:
   - Reconstruct model via `_build_model(algebra_enum, task_name, config)`
   - Load checkpoint: `model.load_state_dict(torch.load(ckpt_path, weights_only=True))`
   - Build task data: `train_ds, test_ds, _ = _build_task_data(task_name, config, seed=42)`
   - Use a data subset for Hessian: `n_hessian = min(200, len(test_ds))`, extract tensors `test_ds.tensors[0][:n_hessian]` and `test_ds.tensors[1][:n_hessian]`
   - Call `compute_hessian_spectrum(model, loss_fn, hessian_x.to(device), hessian_y.to(device), device=device)`
   - Convert numpy arrays to lists: `key = "eigenvalues" if "eigenvalues" in spectrum else "ritz_values"; spectrum[key].tolist()`
   - Store as `hessian_spectrum[f"{frac:.1f}"] = eigenvalue_list` (use 1 decimal place: "0.0", "0.2", "0.5", "1.0")
   - Skip if result.json already has `hessian_spectrum` key (idempotent, unless `--force` flag)

4. **Curvature measurement**: For each seed that has a converged checkpoint (frac=1.0):
   - Load converged model
   - Call `measure_curvature(model, loss_fn, data_x.to(device), data_y.to(device), n_directions=config.n_curvature_directions)`
   - Write `curvature` (float, the mean_curvature) and `curvature_detail` (full dict) to result.json
   - Skip if result.json already has `curvature` key (idempotent, unless `--force`)

5. **Gradient variance collection**: For each unique (task, algebra) combination:
   - Build model factory: `lambda: _build_model(algebra_enum, task_name, config).to(device)`
   - Build data from test set (same subset as Hessian)
   - Call `collect_gradient_variance_across_seeds(model_factory, loss_fn, data_x, data_y, seeds=config.seeds[:5], n_steps=10, device=device)`
   - Save to `{results_dir}/{task}/gradient_variance_{algebra}.json`
   - Also update each seed's result.json with `gradient_stats: {"grad_norm_std": cross_seed_variance ** 0.5, "cross_seed_variance": cross_seed_variance}`
   - Skip if gradient_variance file exists (idempotent, unless `--force`)
